Fun with CSS

The goal of this page is to practice using different CSS elements. We will explore the use of classes, divs, spans, floats, and more. One of the difficulties of developing web pages to practice on is coming up with content to present. It may not end up being the prettiest site, but it will be good practice.

Classes are the first element that we will be presenting. A class is used in CSS to effect specific sections of the presentation. More specifically you can tag a specific paragraph or and of the html without effecting every section of the same type.
